# Package index

## Get started

Essential functions to begin using mobdb

- [`mobdb_has_key()`](https://jasonad123.github.io/mobdb/reference/mobdb_has_key.md)
  : Check if Mobility Database API token is configured
- [`mobdb_set_key()`](https://jasonad123.github.io/mobdb/reference/mobdb_set_key.md)
  : Set Mobility Database API refresh token
- [`mobdb_browse()`](https://jasonad123.github.io/mobdb/reference/mobdb_browse.md)
  : Load the Mobility Database in browser

## Discover feeds

Search and explore available transit feeds

- [`mobdb_search()`](https://jasonad123.github.io/mobdb/reference/mobdb_search.md)
  **\[experimental\]** : Search for feeds across the Mobility Database
- [`feeds()`](https://jasonad123.github.io/mobdb/reference/feeds.md) :
  List and filter feeds

## Download feeds

Download feeds for analysis

- [`download_feed()`](https://jasonad123.github.io/mobdb/reference/download_feed.md)
  : Download GTFS Schedule feed
- [`mobdb_read_gtfs()`](https://jasonad123.github.io/mobdb/reference/mobdb_read_gtfs.md)
  **\[superseded\]** : Read GTFS feed directly from Mobility Database

## Assess feed quality

Evaluate feed quality using MobilityData validation reports

- [`filter_by_validation()`](https://jasonad123.github.io/mobdb/reference/filter_by_validation.md)
  : Filter feeds or datasets by validation quality
- [`get_validation_report()`](https://jasonad123.github.io/mobdb/reference/get_validation_report.md)
  : Get validation report for feeds or datasets
- [`view_validation_report()`](https://jasonad123.github.io/mobdb/reference/view_validation_report.md)
  : View validation report in browser

## Historical datasets

Access and explore historical feed versions

- [`mobdb_datasets()`](https://jasonad123.github.io/mobdb/reference/mobdb_datasets.md)
  : Get datasets for a feed
- [`mobdb_get_dataset()`](https://jasonad123.github.io/mobdb/reference/mobdb_get_dataset.md)
  : Get details for a specific dataset

## Work with feed metadata

Extract and manipulate feed information

- [`mobdb_extract_datasets()`](https://jasonad123.github.io/mobdb/reference/mobdb_extract_datasets.md)
  : Extract latest dataset information from search results
- [`mobdb_extract_locations()`](https://jasonad123.github.io/mobdb/reference/mobdb_extract_locations.md)
  : Extract location information from search results
- [`mobdb_extract_urls()`](https://jasonad123.github.io/mobdb/reference/mobdb_extract_urls.md)
  : Extract download URLs from feed results
- [`mobdb_get_feed()`](https://jasonad123.github.io/mobdb/reference/mobdb_get_feed.md)
  : Get details for a specific feed
- [`mobdb_feed_url()`](https://jasonad123.github.io/mobdb/reference/mobdb_feed_url.md)
  : Get download URL for a feed
